# AI Music Singer 🎤

Transforme letras em música usando inteligência artificial! Digite uma letra, escolha um estilo musical e ouça sua música sendo cantada em tempo real.

## Funcionalidades

- **7 Estilos Musicais**: Pop, Rock, Funk, Rap, Sertanejo, Eletrônica e Gospel
- **Síntese de Voz**: Utiliza Web Speech API para cantar as letras
- **Controle Completo**: Play, Pause, Stop, Volume e Velocidade
- **Histórico**: Suas criações ficam salvas no navegador
- **Exportar TXT**: Baixe suas letras em arquivo de texto
- **Responsivo**: Funciona em desktop e mobile
- **Tema Escuro**: Interface moderna com design noturno

## Como Usar

1. **Digite a letra** no editor de texto
2. **Escolha o estilo musical** (Pop, Rock, Funk, etc.)
3. **Selecione a voz** disponível no seu navegador
4. **Ajuste velocidade e volume**
5. **Clique em "🎵 Cantar"** e ouça sua música
